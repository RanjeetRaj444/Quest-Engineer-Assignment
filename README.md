## How to run locally in your system.

- Clone this repo

```bash
git clone https://github.com/RanjeetRaj444/Quest-Engineer-Assignment.git
```

- Navigate to main folder

```bash
cd myapp
```

- Install dependencies

```bash
npm install
```

- Start server locally

```bash
npm run dev
```
